# Jingle Jam - GDG on Campus TED University

This repository contains the official website for the **Jingle Jam** coding competition organized by **Google Developer Groups (GDG) on Campus TED University**.

## Overview

The website features:
- A countdown timer to the event date (December 20, 2025).
- A festive, responsive design with winter-themed animations.
- An integrated registration form connected to Google Sheets.

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Apps Script (for form backend)
